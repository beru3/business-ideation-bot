## 敗因（可能性の高い順）

1. **検索意図とFVの断絶（主因）**: 広告KWは「カスハラ 対策」系。クリックする人の大半は「義務化で何をすればいいか知りたい」**準備ゼロ〜情報収集段階**。ところがFVは「規程を作って、終わりにしていませんか。問われるのは運用の証跡」——**規程を作り終えた人**にしか刺さらない一段先の訴求。141人の大半は「自分向けではない」と判断して即離脱した
2. **信頼要素ゼロ**: 運営者情報・特商法・プライバシーポリシー・実績のいずれもなし。さらに「※価格は開発中の想定です」とサービス実体の不在を自己申告。B2Bの総務・経営者がメールを渡す条件を満たしていない
3. **オファーが汎用PDF**: 「5本柱チェックリスト」は中身のプレビューがなく、厚労省「あかるい職場応援団」の無料マニュアルと区別がつかない。ナレッジID#1219（診断型CTA）を採用したはずが、実装は「PDF配布」に落ちていた
4. （補助）オーガニック流入の不在: X フォロワー3・imp 18、note スキ10。広告以外の検証チャネルが機能していない

## 較正上の意味（戦略レビューの問いへの回答）

「(a)需要不在 か (b)LP不良 か」——現データでは**(b)が濃厚で(a)は未判定**。141人がLPを見る前に広告をクリックした事実は需要側の弱い肯定。LP改修後の再テストが(a)の判定実験になる。

## 対策（LP改修 v2 — 2026-07-23実装）

1. **FVを検索意図に合わせる**: 「規程を作って終わりにしていませんか」→「2026年10月1日、カスハラ対策が全事業主に義務化。準備はできていますか？」。準備ゼロ層を正面で受け止める
2. **リードマグネットをPDF配布→ブラウザ内3分診断に変更**（ID#1219の正しい実装）: 5本柱×2問=10問のチェック→未対応項目をその場で表示→「未対応項目の対応手順つき完全チェックリスト」をメールで受け取る。メール入力前に価値を先渡しし、CTAの心理障壁を下げる
3. **信頼要素の追加**: 運営者表記・プライバシー方針の明記。「開発中の想定です」→「先行登録優待」フレームに書き換え（事実は変えず表現を反転）
4. **チェックリストの中身プレビュー**を掲載（25項目中5項目を見せる）
