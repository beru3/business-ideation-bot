# カスハラ防止義務化、中小企業が10月までにやること

2026年10月、カスタマーハラスメント防止措置が義務化されます。「カスハラ対策って具体的に何をすればいいの？」という中小企業の経営者・管理部門の方向けに、10月までにやるべきことを整理しました。対応を後回しにすると、行政指導や企業名公表のリスクもあります。今から動けば十分間に合います。

## 1. カスハラ防止義務化とは何か

労働施策総合推進法の改正により、事業者に「カスタマーハラスメント防止のための措置」が義務づけられます。

ポイントは以下の通りです。

- **従業員規模に関わらず、全事業者が対象**
- 顧客等からの著しい迷惑行為から従業員を守る措置が必要
- 違反した場合、**行政指導**や**企業名公表**の対象になり得る
- 施行日は**2026年10月**

「大企業の話でしょ？」と思われがちですが、中小企業も例外ではありません。

## 2. 中小企業が「後回し」にしがちな3つの理由

多くの中小企業がまだ動けていない背景には、共通の誤解があります。

- **「うちはクレームが少ないから大丈夫」** → 義務化は件数に関係なく全事業者が対象です
- **「カスハラの定義が曖昧で、何を対策すればいいか分からない」** → 厚労省が判断基準を公表済みです
- **「マニュアルを作る時間もノウハウもない」** → ひな形を活用すれば1日で着手できます

「分からないから動けない」が最大のリスクです。まずは全体像を把握しましょう。

## 3. 10月までにやるべき5つのこと

具体的なステップは以下の5つです。**順番に進めれば、小規模な会社でも対応可能**です。

1. **カスハラの定義と判断基準を社内で共有する** — 厚労省の指針をベースに「どこからがカスハラか」を明文化
2. **対応マニュアルを策定する** — 厚労省のひな形をダウンロードし、自社に合わせて修正
3. **従業員向け研修を実施する** — 録画形式でOK、1回30分程度で十分
4. **相談窓口を設置する** — 既存のハラスメント相談窓口との兼用も可能
5. **記録・報告フローを整備する** — 発生時の記録テンプレートと報告ルートを決めておく

完璧を目指す必要はありません。まず「着手した」という事実が、行政対応上も重要です。
