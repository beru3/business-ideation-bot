## 3. CTA/価格テクニック

### ID#1219（診断ツール型CTA / confidence 0.85）
- **原文**: 中小企業経営者向け「活用度診断・ロードマップ自動生成」。業種を入力すると優先順位を自動生成
- **適用方法**: CTA本体。「カスハラ義務化対応チェックリスト診断」（無料）→結果ページでウェイトリスト登録。社労士向けは「顧問先の運用カバー率診断」

### ID#1233（無料→有料転換 / confidence 0.85）
- **原文**: 「無料地獄」回避。無料コンテンツの役割を「問題の自覚を生むこと」に限定し、有料への導線を設計
- **適用方法**: 無料診断は「未対応項目の自覚」を生むだけに留め、解決（証跡・台帳・自動追従）は有料側に置く。診断結果ページで欠落項目を赤表示→CTA

### ID#1250（リード品質 / confidence 0.85）
- **原文**: 無料オファーは「タダ乗り客」を集める。低価格有料オファーで見込み客の質を上げる
- **適用方法**: フォームの「有料でも使いたいか」設問で質を測る（Step 14のGo/No-Go指標=有料意向30%）。東京都奨励金パック（高単価買い切り）は質の高い層への別オファー

### ID#1216（アンカリング・心理配置 / confidence 0.9）
- **原文**: アンカリング・フレーミング・社会的証明を販売ページに組み込む（価格表示順序・レビュー配置・CTA文言）
- **適用方法**: LP価格セクション。「社労士に研修＋マニュアル作成を頼むと20〜50万円」を先に提示→月額を対比。社労士向けは「顧問先1社あたり¥980」の単価分解フレーム

### ID#1226 / ID#1257（デコイ価格設計）
- **原文**: 3プラン構成で囮選択肢を置き、中〜高単価プランを選ばせる
- **適用方法**: 価格表は3段（保管専用プラン＝低価格デコイ兼解約受け皿 / 標準 / 奨励金対応パック）。標準を「おすすめ」に

### ID#1237（LP視線設計 / confidence 0.85）
- **原文**: F動線・Z動線に基づくCTA配置・見出し設計
- **適用方法**: LPはFV直下に第1CTA、Z動線終点（右下）にもCTA。チェックリスト診断ボタンは画面遷移ごとに固定表示

### ID#1230（B2Bリードスコアリング / confidence 0.85）
